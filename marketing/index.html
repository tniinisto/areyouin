Set and see team's or group's events & statuses easily

It all started from a group of friends playing beach volleyball and the fact that setting the game time that would suite everyone was sometimes difficult! There were multiple mails, messages and phone calls to set the time that would suite. During this came the idea that an application would help with this. The first version came to usage around summer 2012 and after that we have been using it. During the years it has developed further and features that have been seen usefull were added. The main principal has been on the ease of usage.

It is a responsive web application that works basically on all modern web browsers on pc, tablets and smartphones and requires no installations. The fixes and new features come available for everybody automatically. The main functionality is on the events and participants who can easily set them IN or OUT. Adding a new event is quick and easy and also on new events there is a optional email notification send to the participants with the basic event information in it. Event has start & end time, location and of course the participants. On Chrome and Firefox web browser notifications are currently supported with pc and tablet usage. With these you will be notified on event changes and new chat messages when you are logged in and even if the browser in minimised. The chat is available on all team's members where it is easy to get the message for everyone to see it. Event location has also a weather information with it and this can be shown on the event page if wanted by location. Location can be seen on Google Maps. On top of these there is also the user's profile which she/he can edit. User can also see activity charts on how many events the has been and how many she/he has attended.
